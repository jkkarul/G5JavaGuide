Lesson Module:

Learning Objectives

After completing this lesson, you should be able to:

1. Define expressions and understand their components (operands and operators).

2. Perform basic arithmetic operations using expressions in Java.

3. Increment and decrement variables using the shorthand operators.

4. Understand integer division and its behavior in Java.

5. Use type casting to convert data types in expressions.

Lesson Content

1. Introduction to Expressions

• An expression is a combination of operands and operators.

• Operands are values, variables, numbers, or quantities used in an expression.

• To write programs, we use programming languages.

• Operators are arithmetic symbols (+, -, *, /, %) that perform operations on operands.

2. Arithmetic Operations with Expressions

• Addition:

int friends = 10; friends = friends + 1; // Output: 11 JAVA

• Subtraction:

friends = friends - 1; // Output: 10 JAVA

• Multiplication:

friends = friends * 2; // Output: 20 JAVA

• Division:

friends = friends / 2; // Output: 5 JAVA

• Modulus (remainder of division):

friends = 10 % 3; // Output: 1 JAVA

3. Increment and Decrement Operators

• Increment operator (++) adds 1 to a variable:

friends++; // Output: 11 JAVA

• Decrement operator (--) subtracts 1 from a variable:

friends--; // Output: 9 JAVA

4. Integer Division

• When dividing integers, Java truncates the decimal portion (remainder) of the result.

• Example:

int result = 10 / 3; // Output: 3 (instead of 3.33) JAVA

• Solution: Use type casting to convert the result to a floating-point data type.

double result = (double) 10 / 3; // Output: 3.3333333333333335 JAVA

5. Type Casting in Expressions

• Type casting is used to convert a value from one data type to another.

• Syntax: (TargetDataType) valueToBeConverted

• Example:

double friendsDouble = (double) friends; // Convert int to double JAVA

Lesson Module:

Learning Objectives

After completing this lesson, you should be able to:

1. Understand the purpose and usage of the Math class in Java.

2. Utilize various methods of the Math class for common mathematical operations.

3. Implement user input and perform calculations using the Math class methods.

4. Create a program to find the hypotenuse of a triangle using the Math class.

Lesson Content

1. Math Class Methods:

• Math.max(x, y): Returns the larger of two values.

let a = 5;
let b = 10;
let maxValue = Math.max(a, b); // maxValue will be 10 JAVA

• Math.min(x, y): Returns the smaller of two values.

let c = 7;
let d = 3;
let minValue = Math.min(c, d); // minValue will be 3 JAVA

• Math.abs(x): Returns the absolute value of a number.

let e = -8;
let absoluteValue = Math.abs(e); // absoluteValue will be 8 JAVA

• Math.sqrt(x): Returns the square root of a number.

let f = 16;
let squareRoot = Math.sqrt(f); // squareRoot will be 4 JAVA

• Math.round(x): Returns the closest integer to the argument.

let g = 3.14;
let roundedValue = Math.round(g); // roundedValue will be 3 JAVA

• Math.ceil(x): Returns the smallest (closest to negative infinity) value that is greater than or equal to the argument and is equal to a mathematical integer.

let h = 3.14;
let ceiledValue = Math.ceil(h); // ceiledValue will be 4 JAVA

• Math.floor(x) Returns the largest (closest to positive infinity) value that is less than or equal to the argument and is equal to a mathematical integer.

let i = 3.14;
let flooredValue = Math.floor(i); // flooredValue will be 3 JAVA

2. Sample Code Using Math Class - Calculating the Hypotenuse:

• The hypotenuse of a right-angled triangle is the longest side of the triangle, opposite the right angle.

• The formula to calculate the hypotenuse is: c = sqrt(a^2 + b^2), where a and b are the lengths of the other two sides.

• Sample Code: 

import java.util.Scanner; // Import the Scanner class to read user input

public class HypotenuseCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); // Create a new Scanner object to read user input
                        
        // Prompt the user to enter the length of side x
        System.out.print("Enter side x: ");
        double x = scanner.nextDouble(); // Read the user's input for side x
                        
        // Prompt the user to enter the length of side y
        System.out.print("Enter side y: ");
        double y = scanner.nextDouble(); // Read the user's input for side y
                        
        // Calculate the length of the hypotenuse using the Pythagorean theorem
        double z = Math.sqrt(x * x + y * y);
                        
        // Print the calculated value of the hypotenuse
        System.out.println("The hypotenuse is: " + z);
                        
        scanner.close(); // Close the Scanner object to free up system resources
    }
}

Lesson Module:

Learning Objectives

After completing this lesson, you should be able to:

1. Import the java.util.Random class to access random value generation methods.

2. Create an instance of the Random class to generate random integers, doubles, and booleans.

3. Generate random integers within a specified range.

4. Generate random double values between 0.0 and 1.0.

5. Generate random boolean values (true or false).

Lesson Content

1. Importing the Random Class

• To use random values in Java, we need to import the java.util.Random class. We'll do this at the top of our class, outside of the main method: 

import java.util.Random; JAVA

2. Creating a Random Instance

• Next, we'll create an instance of the Random class within the main method. This will allow us to access the methods for generating random values:

Random random = new Random(); JAVA

• System.in represents the standard input stream, which is typically the console or command-line input.

3. Generating Random Integers

• To generate a random integer, we can use the nextInt() method of the Random class. By default, this will generate a random integer between Integer.MIN_VALUE and Integer.MAX_VALUE:

int x = random.nextInt();
System.out.println("Random integer: " + x); JAVA

• If we want to limit the range of the random integer, we can pass an argument to the nextInt() method. For example, to generate a random number between 1 and 6 (like rolling a six-sided die), we can do:

int diceRoll = random.nextInt(6) + 1;
System.out.println("Random dice roll: " + diceRoll); JAVA

4. Generating Random Doubles

• To generate a random double value between 0.0 and 1.0, we can use the nextDouble() method:

double y = random.nextDouble();
System.out.println("Random double: " + y); JAVA

5. Generating Random Booleans

• Finally, to generate a random boolean value (either true or false), we can use the nextBoolean() method:

boolean z = random.nextBoolean();
System.out.println("Random boolean: " + z); JAVA
